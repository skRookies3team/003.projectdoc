### 안건 1: [기능 구체화 & 구현 가능성 검토]
- **논의 내용**: 
  - AI 사진업로드 + 일기 생성, 디지털 페르소나 챗봇, 일대기 생성 기능이 구체적으로 어떻게 동작하는지 의논하고, 구현 가능성을 조사
    
- **결정 사항**: 
  - 이미지 & 영상 API + 기술 구체화 추가 검토 필요
  - 디지털 페르소나 챗봇 : RAG Gemini API의 File Search 활용
  - AI 사진 업로드 + 일기생성 : 구현가능성, 기술스택, 시장현황, 환경구축 조사 (검토 필요)
- **담당자**: 이가은, 김지은, 신재석 
- **마감일**: 2025-11-11

### 안건 2: 문제 정의
- **논의 내용**: 
   - 프로젝트 방향성 확립을 위한 프로젝트 배경 조사
- **결정 사항**
  - 프로젝트 배경

    1. 반려동물 시장 현황 및 성장성
    2. 기록 및 정보 관리의 어려움/기록 작성의 귀찮음 문제
    3. 반려동물 콘텐츠 소비 증가
    4. 펫로스(AI 기술으로 모든 사람의 소중한 추억을 자동으로 보존하고 공유하는 서비스를 제공)

  * 핵심 기능 및 해결 포인트
    - **기록의 귀찮음**: 매일 수동으로 일기를 작성하는 것이 번거로움 → **AI 자동 일기 생성**으로 해결
    - **사진 정리의 어려움**: 수천 장의 사진이 쌓이지만 정리와 의미 부여가 어려움 → **자동 아카이빙 및 분류**로 해결
    - **건강 관리의 어려움**: 반려동물이 아플 때 대처가 힘들고 정보가 부족함 → **건강 기록 및 이상 감지 기능**으로 해결(근데 이거는 안할 거같음)
    - **추억의 휘발성**: 시간이 지나면 당시의 감정과 상황을 잊어버림 → **AI 기반 맥락 있는 기록**으로 해결
    - **준비 부족**: 입양 준비가 충분하지 않았다고 느끼는 비율이 71.6% → **정보 제공 및 커뮤니티**로 해결

- **담당자**: 손민정, 양승준 
- **마감일**: 2025-11-11
