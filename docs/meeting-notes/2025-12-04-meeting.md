## 회의 내용

### 안건 1: 금주 핵심 작업 할당 및 역할 분담 (To-Do 확정)

* **결정 사항**: 아래 표와 같이 작업을 할당하고 진행한다.

| 담당자 | 핵심 작업 | 비고 |
| :--- | :--- | :--- |
| **양승준** | 🤖 AI 아키텍처 그리기 및 환경 구축, 챗봇 기본 구조 구현, FRONTEND UI 수정 및 **FASD 구조** 정하기 | AI 기능 개발 총괄 |
| **김지은** | 📷 다이어리 이미지 출처에 따른 **저장 로직 분기 처리**, 이미지 중복 전송 방지 로직 구현 | 이미지 처리 로직 중요 |

### 안건 2: 프론트엔드 구조 및 UI 기능 개발 계획
* **논의 내용**: 신재석 팀원이 Next.js를 React/Vite로 변환하는 작업 후, 프론트엔드 UI 기능 추가 및 양승준 팀원의 UI 수정 작업을 조율. 양승준 팀원은 **FASD (Frontend Architecture Design)** 구조를 정의하여 일관성을 확보하기로 함.
* **결정 사항**:
    * 신재석 팀원은 UI 기능 추가 및 수정 작업을 집중적으로 진행한다.
    * 양승준 팀원은 프론트엔드 UI 수정 작업과 함께 **FASD 구조를 정의**하여 팀원들에게 공유한다.

### 안건 3: AI 서버 아키텍처 및 환경 구축 방안
* **논의 내용**: 리캡 기능 구현을 위해 AI 서버 환경 구축이 필수적이며, 아키텍처를 빠르게 확정해야 함.
* **결정 사항**:
    * 양승준 팀원이 **AI 아키텍처 다이어그램**을 그리고, 개발 환경을 구축한 후 챗봇의 기본 구조(인터페이스)를 마련한다.

### 안건 4: 클라우드 환경 구축 및 문서 작업 진행 상황 점검
* **논의 내용**: 클라우드 사용 계획서 작성 및 제출 필요성 점검.
* **결정 사항**:
    * 손민정 팀원은 **클라우드 사용 계획서**를 작성하고 제출하여 인프라 구축의 다음 단계를 진행할 수 있도록 한다.
