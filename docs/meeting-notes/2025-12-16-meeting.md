### 안건 2: Claude Skills를 활용한 AI 로직 재구성 및 최적화
* **논의 내용**: 기존 AI 호출 로직을 Claude Skills 파일 기반의 도구 사용(Tool Use) 로직으로 변경하여 서비스 응답 효율과 정확도를 높이는 방안 논의.
* **결정 사항**:
    * **Claude Skills** 로직을 프로젝트 구조에 맞게 이식하여 AI 답변 및 다이어리 생성 로직을 재구성한다.
    
* **담당자**: 양승준
* **마감일**: 2025-12-26

### 안건 3: 프론트엔드 로그인/회원가입 로컬 테스트 및 다이어리 CRUD 연동
* **논의 내용**: 로컬 환경에서 회원가입 및 로그인 기능이 정상 작동하는지 교차 검증하고, 다이어리 서비스의 프론트 연동 진행 상황 점검.
* **결정 사항**:
    * **로컬 실행 테스트**를 마무리하고, 현재 진행 중인 **다이어리 CRUD 프론트 연동** 작업을 가속화한다.
    * 프론트엔드 단에서의 **로그인 기능 테스트**를 병행하며 백엔드 추가 요구 기능을 개발한다.
* **담당자**: 김지은, 김현근
* **마감일**: 2025-12-26

### 안건 4: 인프라 보안 그룹(SG) 수정 및 API Gateway 배포 설정 점검
* **논의 내용**: EKS Node Group의 통신 보안을 위한 Terraform 코드 수정과 운영 환경(prod) 배포를 위한 설정 파일(yaml) 작성 논의.
* **결정 사항**:
    * Terraform을 통해 **Node Group SG**를 최신화하고, **application-prod.yaml** 파일을 작성하여 API Gateway 서비스를 성공적으로 배포한다.
    
* **담당자**: 손민정
* **마감일**: 2025-12-26
