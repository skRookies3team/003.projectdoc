## 회의 내용

### 안건 1: 금주 핵심 작업 할당 및 역할 분담 (To-Do 확정)
* **논의 내용**: 소셜 피드 공유 기능 고도화, AI 실시간 통신 구현, 코인 시스템 수정 등 기능 완성 단계의 작업들을 팀원별로 할당함.
* **결정 사항**: 아래 표와 같이 작업을 할당하고 진행한다.

| 담당자 | 핵심 작업 | 비고 |
| :--- | :--- | :--- |
| **양승준** | AI 페르소나 챗봇 실시간 동기화 구현, JavaScript SSE 통신 테스트 | AI 실시간 연동 담당 |
| **김지은** | AI 다이어리 소셜 공유 및 공개 범위 설정 구현, 이미지 전송 방식 변경(JSON), S3 업로드 자동화 | 이미지/공유 로직 총괄 |
| **이가은** | 코인 적립 및 사용 API 수정, 코인 이용 내역 조회 API 신규 구현 | 재화 시스템 고도화 |
| **신재석** | 펫메이트 프론트엔드 및 백엔드 잔여 에러 수정 | 서비스 안정화 |
| **김현근** | 소셜 서비스 API 연동 및 UI 수정, 이미지 업로드 로직 리팩토링 | 소셜 기능 최적화 |
| **손민정** | Postman을 통한 ALB DNS 외부 통신 테스트 진행 | 인프라 가용성 검증 |

### 안건 2: AI 페르소나 챗봇 실시간 동기화 및 SSE 기술 적용
* **논의 내용**: 사용자와 챗봇 간의 대화 흐름을 실시간으로 반영하기 위해 JavaScript 기반 Server-Sent Events(SSE) 효율성 검증 및 적용 방안 논의.
* **결정 사항**:
    * SSE 테스트 완료 후 페르소나 챗봇의 응답을 실시간으로 프론트엔드에 동기화하는 로직을 최종 구현한다.

* **담당자**: 양승준

### 안건 3: 다중 이미지 처리 로직 개선 및 S3 자동화 인프라 점검
* **논의 내용**: 일기 생성 시 다중 이미지를 S3에 자동 업로드하고, 이를 보관함 서비스와 유기적으로 연동하는 아키텍처 논의.
* **결정 사항**:
    * 회원 서비스의 보관함과 S3를 연동하여 일기 이미지 업로드를 자동화하고 다중 저장 로직을 완성한다.

* **담당자**: 김지은

### 안건 4: 소셜 피드 업로드 방식 변경 및 미리보기 메모리 관리 최적화
* **논의 내용**: 기존 Multipart 방식의 업로드를 JSON 기반으로 변경하여 데이터 규격을 통일하고, 다중 이미지 미리보기 시 발생하는 Blob 메모리 누수 해결 방안 논의.
* **결정 사항**:
    * 업로드 방식을 JSON으로 전환하고 Blob 관련 메모리 관리 오류를 수정한다.
    * 소셜 서비스의 UI를 수정하고 이미지 업로드 백엔드 로직을 이에 맞춰 리팩토링한다.
* **담당자**: 김지은, 김현근

### 안건 5: 코인 시스템 API 고도화 및 인프라 외부 통신 테스트
* **논의 내용**: 사용자의 활동에 따른 코인 적립/사용 로직의 정확성 검증과 인프라의 로드밸런서(ALB) 통신 상태 확인.
* **결정 사항**:
    * 코인 적립/사용 API의 비즈니스 로직을 수정하고 내역 조회 기능을 추가한다.
    * ALB DNS를 통해 실제 외부 통신이 정상적으로 이루어지는지 Postman으로 최종 테스트한다.
* **담당자**: 이가은, 손민정
