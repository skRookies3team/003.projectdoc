## 회의 내용

### 안건 1: 금주 핵심 작업 할당 및 역할 분담 (To-Do 확정)
* **논의 내용**: Gateway 연동, 마일리지 시스템 구현, 인프라 구축 등 핵심 백엔드/인프라 작업 중심으로 역할 분담 및 집중도를 높이기로 함. 
* **결정 사항**: 아래 표와 같이 작업을 할당하고 진행한다.

| 담당자 | 핵심 작업 | 비고 |
| :--- | :--- | :--- |
| **김지은** |  **Gateway 연동** 및 **JWT 인증** 기반 Diary Service 라우팅 구현, 다이어리 **Style Widget** 설정 및 관리 API 구현 | 인증/라우팅 및 Diary 핵심 API |
| **이가은** |  **마일리지 조회, 적립, 차감 기능 구현**, **S3 사진 업로드 기능** 구현 | 마일리지 시스템 및 S3 통합 |
| **신재석** |  **프론트 UI 기능 추가 및 수정**, **펫메이트 백엔드 API 연동** 구현 | 프론트엔드 구현 및 연동 |
| **양승준** |  **FE FASD 구조 변환 지원** 및 **AXIOS 통신 파일 생성** 지원, **OpenAI/Bedrock API 키 연동 확인 및 테스트** | FE 구조 지원 및 AI 연동 테스트 |
| **손민정** |  **VPC 설정**, **Terraform 설치** 및 **EKS Cluster 생성** | 핵심 인프라 구축 |
| **김현근** | **Gateway 연동** 및 **JWT 인증** 기반 Social Service 라우팅 구현, 소셜 ADR 문서 작성, 해시태그 기능 추가, API 게이트웨이 통신 확인 |

### 안건 2: 백엔드 서비스 연동 및 인증(JWT) 전략 논의
* **논의 내용**: 마이크로서비스 간의 통신을 Gateway로 일원화하고, JWT를 기반으로 각자의 서비스의 접근을 관리하는 방안 논의.
* **결정 사항**:
    * **Gateway 연동**을 진행하고 **JWT 인증**을 통해 각자의 서비스로의 라우팅을 구현한다.
    * 각자의 서비스에 필요한 **Style Widget 설정/관리 API**를 구현한다.
* **담당자**: 김지은, 김현근
* **마감일**: 2025-12-12

### 안건 3: 마일리지 시스템 및 S3 파일 업로드 구현 계획
* **논의 내용**: 마일리지의 정확한 적립/차감 로직과 S3를 통한 파일 업로드 기능 구현 방안 논의.
* **결정 사항**:
    * **마일리지 CRUD (조회, 적립, 차감)** 기능을 구현하여 핵심 비즈니스 로직을 완성한다.
    * S3 사진 업로드 기능을 구현한다.
* **담당자**: 이가은
* **마감일**: 2025-12-12

### 안건 4: AI API 키 연동 및 프론트엔드 구조 지원 방안
* **논의 내용**: AI 모델(OpenAI, Bedrock)의 API 키 연동 테스트 및 프론트엔드의 구조적 안정화 지원 방안 논의.
* **결정 사항**:
    * **FE FASD 구조 변환**을 지원하고, 프론트엔드 팀이 백엔드와 쉽게 통신할 수 있도록 **AXIOS 통신 파일 생성**을 돕는다.
    *  AI 기능 구현을 위해 **OpenAI 및 AWS Bedrock API 키 연동을 확인하고 테스트**를 완료한다.
* **담당자**: 양승준
* **마감일**: 2025-12-12

### 안건 5: 인프라 구축 (Terraform, EKS) 계획 확정
* **논의 내용**: Terraform을 사용하여 AWS 인프라(VPC, EKS Cluster)를 코드로 관리(IaC)하는 방안 논의.
* **결정 사항**:
    *  **VPC 설정**을 시작하고, **Terraform을 설치**하여 **EKS Cluster 생성**까지 인프라 구축 작업을 진행한다.
* **담당자**: 손민정
* **마감일**: 2025-12-12
