### 안건 2: AI 이미지-3D 모델 변환 기술 검토 및 에이전트 도입 논의
* **논의 내용**: Microsoft의 Image to 3D 기술을 활용하여 2D 이미지를 3D 모델로 자동 생성하는 프로세스와 오픈소스 렌더링 적용 현황 논의. KRAFTON-ai 에이전트 도입 여부 및 Slack/텔레그램 연동 가능성 검토.
* **결정 사항**:
    * 오픈소스를 활용한 3D 모델 자동 렌더링 적용을 지속하며, 에이전트 도입 실효성을 판단하여 다음 회의 시 공유한다.

* **담당자**: 양승준
* **마감일**: 2025-12-26

### 안건 3: 백엔드 이미지 처리 로직 수정 및 PostGIS 인프라 고도화
* **논의 내용**: Swagger 테스트 환경에서의 이미지 업로드 오류 해결과 위치 기반 서비스 강화를 위한 Docker 환경 내 PostGIS 삽입 논의.
* **결정 사항**:
    * Swagger 테스트 시 발생하는 이미지 삽입 오류를 수정하고, Dockerfile에 PostGIS 설정을 추가하여 공간 데이터 처리 환경을 구축한다.

* **담당자**: 김지은
* **마감일**: 2025-12-26

### 안건 4: 프론트엔드 마이페이지 및 실시간 데이터 연동 최적화
* **논의 내용**: 마이페이지의 펫 정보 동기화 이슈 해결과 AI 일기 생성 페이지 내 실시간 위치/펫 목록 연동 방안 논의.
* **결정 사항**:
    * 마이페이지 펫 생성 시 데이터가 즉각 반영되도록 수정하고, 프록시 요청 설정을 점검하여 통신 오류를 방지한다.
    * AI 일기 생성 페이지에서 사용자의 위치와 펫 목록이 실시간으로 연동되도록 통합 작업을 수행한다.
* **담당자**: 김지은
* **마감일**: 2025-12-26

### 안건 5: 보관함 API 연동 및 소셜 서비스 이미지 업로드 로직 개선
* **논의 내용**: 사용자 보관함의 사진 관리 기능 구현과 소셜 서비스 내 대용량 이미지 업로드 처리 로직 개선 논의.
* **결정 사항**:
    * 보관함 사진 조회 및 업로드 API의 프론트엔드 연동을 완료한다.
    * 소셜 서비스의 이미지 업로드 로직을 수정하여 서버 부하를 줄이고 UI와의 연동성을 높인다.
    * 펫메이트 서비스에 키워드 기반 위치 검색 기능을 추가하여 편의성을 개선한다.
* **담당자**: 이가은, 김현근, 신재석
* **마감일**: 2025-12-26
